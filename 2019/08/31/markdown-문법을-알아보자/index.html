Markdown?

Markdown이라는 언어를 들어보셨나요? 아마 들어는 보셨더라도 익숙하지 않으니 이 포스팅을 보고 계실 거라 생각합니다.
저 또한 github Page를 준비하면서 처음 들어본 언어였는데 몇 번 TEST를 하면서 "쉽다"라는 느낌을 많이 받았습니다.
물론 네이버 블로그에서 포스팅하는 것보다는 어려워서 많이 헤맸지만요!
간단하게 먼저 Markdown의 정의에 대해 짚어 봅시다.

Markdown은 일반 텍스트 문서에 텍스트 서식 을 추가하는 데 사용할 수 있는 간단한 마크 업 언어이다.

일단 이 정도까지만 알아도 Markdown을 사용하는 데는 전혀 문제가 없으니 이제 실전으로 넘어갑시다.
인생은 언제나 실전.

Markdown에서 HTML 문법 사용이 가능하며 확장자는 .md 또는 .markdown 입니다.


제목(Headings)

#다음에 띄어쓰기를 넣어야 제목으로 적용됩니다.


참고로 #의 개수에 따라 단순히 글자의 크기만 바뀌는 것이 아니고, #의 개수가 많을수록 상위 제목에 종속이 된다는 점입니다. 예를 들면 밑에 와 같이 입력했을 때, ### 제목 3은 ## 제목 2의 소제목이 됩니다.
## 제목 2
### 제목 3


강조(Emphasis)

  • 출력
    안녕하세요. 반갑습니다.
    안녕하세요. 반갑습니다.
    안녕하세요. 반갑습니다.
    안녕하세요. 반갑습니다.

강조 표시 여러 개를 동시에 설정할 수 있습니다.
**_이탤릭체와 굵기_**
이탤릭체와 굵기
**<u>밑줄과 굵기<u/>**
밑줄과 굵기
**<u>*이탤릭체*와 밑줄과 굵기<u/>**
이탤릭체와 밑줄과 굵기

바로 이렇게요.

인용문 1

인용문 2

인용문 3-1
인용문 3-2

    안녕하세요.

이미지(Images)

HTML의 <img> 태그도 사용 가능합니다.
Markdown에는 이미지 크기를 조절하는 구문이 없기 때문에, 이미지를 조절하고 싶으시면 <img> 태그를 사용하시면 됩니다.

  • 출력



유튜브


수평선(Horizontal Rules)

본문에서 구역을 나누고 싶다면 벌표(*), 하이픈(-), 밑줄(_) 중 하나를 3개 이상 입력하세요.

  • 출력

링크(Links)

  • 출력
    NAVER
    GOOGLE

클릭하시면 네이버로 이동할 수 있습니다!
클릭하시면 1 혹은 2 링크로 이동할 수 있습니다!


표(Tables)

표를 생성하려면 세 개 이상의 하이픈(---)과 파이프(|)를 사용하면 됩니다.

  • 출력
이름 나이 성별
김아무개 20살 여성
최아무개 40살 남성
박아무개 34살 여성
왼쪽정렬 가운데정렬 오른쪽정렬

표를 만드는 게 어려우시면 테이블 생성기의 도움을 받으세요.


각주(Footnotes)

각주를 사용하면 간단하게 메모 및 참조를 추가할 수 있습니다.
각주를 생성하려면 대괄호 안에 캐럿과 식별자를 입력하세요([^1]).
단, 식별자는 숫자나 단어일 수 있지만 공백이나 탭은 불가능합니다.

  • 출력
    지우[1]가 물었다. “피카츄 너는 왜 몬스터 볼에 들어가지 않는 거야?”
    피카츄[2]가 말했다. “피카피카.”

체크리스트(Check Lists)

Markdown 환경에 따라 될 수도, 안 될수도 있습니다.

  • java 공부하기
  • IPv6 포스팅하기
  • 밥 먹기

이스케이프 문자(Escaping Characters)

리터럴 문자를 표시하려면 해당 기호 앞에 이스케이프(\)를 붙이면 됩니다.

  • 출력
  • 하이루

/* 하이루

**이스케이프 적용 가능한 기호**


줄바꿈

Markdown에서 줄바꿈을 하려면 엔터를 누르면 됩니다.
하지만 엔터를 여러 번 눌러보아도 한 칸만 적용되므로 더 띄우고 싶을 땐, 정해진 구문을 입력해야 합니다.

  • 출력

안
녕
하

세

요

출력된 결과물을 보면 알겠지만 '하’와 ‘세’ 엔터 간격은 한 칸, '세’와 '요’의 엔터 간격은 두 칸으로 차이가 나야 하지만, 결과물에는 똑같은 간격으로 출력 되었습니다.

  • 출력

안
녕
하

세


요

<br>을 입력하면 차이가 없던 엔터 간격이 차이가 나는 걸 볼 수 있습니다.
